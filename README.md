# Anti Theft Locker System

- Detects Wrong Attempt
- Detects Force Breach
- Sends Notification on Telegram
- Fully Controllable Via Web UI
- Live Status on Web UI

## Components 
- LCD 16x2 Display
- I2C for LCD
- Servo Motor (x2)
- 4x4 Keypad
- Vibration Sensor SW-420
- Buzzer

# Technologies
- ESP32
- Arduino IDE
- C++
- React with Firebase (for connecting esp)
